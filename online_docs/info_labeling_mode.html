Шаг 1 - Создание данных для разметки

Вначале подготовьте данные для разметки. Для этого запустите docreader в режиме разметки, путем включения строки labeling_mode=True в ваш конфиг файл config.py.

Теперь вы можете создавать данные для разметки путем обычной загрузки ваших файлов. Результаты работы сервиса появятся в дирктории которую вы указале в параметре intermediate_data_path в конфиг-файле
