Word

  1. doc ( result, result in html )
  2. docx ( result, result in html )
  3. odt ( result, result in html )

Excel

  1. ods (openoffice excel) ( result result in html )
  2. xls (old excel format) ( result, result in html )
  3. xlsx (new excel format) ( result, result in html )

PowerPoint

  1. ppt ( result result in html )
  2. pptx ( result, result in html )
  3. odp ( result, result in html )

CSV comma separated values

If you use a non-standard separator, you can pass its value in the "delimiter" parameter into POST-request.
For example delimiter=";". Use cases you can find in dedoc_project/dedoc/tests/test_api_csv.py

  1. csv ( result, result in html )
  2. tsv ( result, result in html )

Others

  1. txt file ( result, result in html )
  2. json file ( result, result in html ) Here, the keys will be analyzed as vertex types, and lists are treated as lists.
